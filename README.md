# FocusRSS Legal and Support Site

This public repository hosts the static legal and support website for FocusRSS.

It contains only the public website, legal-support drafts, and validation/deployment workflow required for GitHub Pages. It does not contain the FocusRSS iOS application source code.

Production domain:

- <https://focusrss.top/>

Important routes:

- <https://focusrss.top/privacy/>
- <https://focusrss.top/terms/>
- <https://focusrss.top/support/>
- <https://focusrss.top/privacy-choices/>

Local validation:

```bash
./scripts/validate-site.sh
```
